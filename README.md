# Sajdah - Islamic Assistant App

Sajdah is a premium Islamic Android application built using modern Android development practices.

## Features
- **Offline Prayer Times:** Accurate calculations using multiple calculation methods.
- **Smart Reminders:** Push notifications for prayer timings, working completely offline.
- **Qibla Compass:** Find the exact direction to the Kaaba using device sensors.
- **Zakat Calculator:** Calculate Zakat efficiently on different assets.
- **Islamic UI/UX:** Material 3, Dark & Light Mode support, custom theming.

## Tech Stack
- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Architecture:** MVVM
- **Dependency Injection:** Dagger Hilt
- **Background Tasks:** WorkManager & AlarmManager
- **Local Data:** DataStore Preferences
- **Library:** Adhan (for prayer calculations)

## Setup
1. Open this project in **Android Studio**.
2. Sync Gradle files.
3. Build and Run on an Android Device or Emulator.

## Permissions Required
- Location (For automatic prayer timings & Qibla calculation)
- Notifications (For prayer reminders)
